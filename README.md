# 📱 Qalbi App - تطبيق قلبي
**تطبيق إسلامي متكامل مبني باستخدام Flutter ويوفر تجربة مستخدم سلسة مع ميزات متعددة**

---

### 🚀 التقنيات المستخدمة (Tech Stack)
| التقنية | الوظيفة |
| :--- | :--- |
| **Flutter & Dart** | الإطار الأساسي للتطوير |
| **Bloc & Cubit** | إدارة حالة التطبيق (State Management) |
| **Dio** | التعامل مع الـ APIs وجلب البيانات |
| **Hive & SharedPrefs** | تخزين البيانات محلياً (Offline Support) |
| **GoRouter** | التنقل السلس بين الشاشات |

---

### ✨ الميزات الرئيسية (Features)
**قراءة القرآن الكريم وسماعه مع التفسير وأحكام التجويد** 📖
**مواقيت الصلاة دقيقة بناءً على موقعك الحالي** 🕌
**تحديد اتجاه القبلة بدقة عالية** 🕋
**أذكار الصباح والمساء والتنبيهات** 📑
**سبحة إلكترونية متطورة مع سجل للتسبيح** 📿
**حاسبة الزكاة لتسهيل المعاملات المالية** 💰
